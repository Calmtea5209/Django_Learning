# Blog Web 部落格網站

## 目前功能
* 擁有登入、登出、註冊系統
* 顯示文章作者與最後修改時間
* 針對登入與否，顯示不同的頁面
* 使用者可自由增加、修改、刪除文章
* 非創建文章之使用者無法修改、刪除文章
* 文章顯示順序依照發佈先後排序

## 開發工具
* VSCode

## 網站主要頁面
```
首頁  /
登入  /accounts/login
登出   /accounts/logout
註冊   /accounts/register
撰寫文章   /write_article
文章內頁   /article/文章id
刪除文章  /article/delete/文章id
修改文章  /article/update/文章id
```
